
# BMI Calculator 

This is a simple Java-based BMI Calculator project developed using Java Swing and MySQL. The project calculates Body Mass Index (BMI) using the user's height and weight. It also includes health-related survey questions and provides basic health suggestions according to the result.

## Features
- Calculate BMI
- User Information Form
- Health Survey Questions
- Smoking, Alcohol, Exercise and Disease Related Questions
- BMI Report and Suggestions
- Simple GUI using Java Swing

## Technologies Used
- Java
- Java Swing
- MySQL
- XAMPP
- Apache NetBeans
